# 경제 뉴스 링크 모음

이 프로젝트는 주요 한국 경제 뉴스 사이트의 RSS 피드를 크롤링하여 최신 경제 뉴스 링크를 수집하고 정리하는 웹 애플리케이션입니다.

## 기능

- 5개 주요 경제 뉴스 사이트의 RSS 피드 크롤링
- 최근 3일 이내 뉴스 자동 수집 (최대 20개)
- 뉴스 카테고리 자동 분류 (증시, 부동산, 금융, 기업, 정책 등)
- 한국어 인코딩 지원
- 회사명 자동 강조
- 반응형 웹 인터페이스
- 실시간 뉴스 업데이트

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip3 install -r requirements.txt
```

2. 웹 애플리케이션 실행:
```bash
python3 app.py
```

3. 브라우저에서 접속:
```
http://localhost:5050
```

## 사용법

1. **메인 페이지**: 수집된 뉴스 링크들을 카테고리별로 확인
2. **뉴스 업데이트**: 상단의 "뉴스 업데이트" 버튼으로 최신 뉴스 수집
3. **기사 읽기**: 각 뉴스 카드의 "기사 읽기" 버튼으로 원문 사이트 이동

## 뉴스 카테고리

- 📈 증시: 주식, 코스피, 코스닥 관련 뉴스
- 🏠 부동산: 아파트, 집값, 전세 관련 뉴스  
- 💰 금융: 금리, 은행, 대출 관련 뉴스
- 🏢 기업: 기업 실적, 매출 관련 뉴스
- 🏛️ 정책: 정부 정책, 규제 관련 뉴스
- 📰 일반: 기타 경제 뉴스

## 자동화 설정

매일 아침 6시에 자동으로 실행되도록 cron job을 설정하려면:

1. crontab 편집:
```bash
crontab -e
```

2. 다음 라인 추가:
```
0 6 * * * cd /path/to/project && /usr/bin/python3 app.py
```

## 기술 스택

- **Backend**: Flask (Python)
- **Frontend**: Bootstrap 5, Bootstrap Icons
- **데이터**: RSS 피드 파싱, JSON 저장
- **크롤링**: feedparser, requests, BeautifulSoup

## 주의사항

- 웹사이트의 robots.txt 정책을 준수합니다.
- 과도한 요청을 피하기 위해 적절한 딜레이를 포함합니다.
- 에러 발생 시 로그를 남깁니다.
- 개발 서버이므로 프로덕션 환경에서는 WSGI 서버 사용을 권장합니다.
